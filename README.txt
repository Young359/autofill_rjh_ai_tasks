写这个项目完全是因为作者受不了这类低质量且浪费时间的所谓教学任务，希望自己的一点小小的工作可以为瑞金的同学们节省一些生命去做更有意义的事情！

程序会被防火墙识别为病毒，选择信任即可。
程序启动之后会自动弹出浏览器，请不要点浏览器内任何元素！尤其是选择科室环节，请在程序命令行里输入数字选择而不是用鼠标选择浏览器内的选项。否则会运行失败。
有时候由于网页本身原因加载慢，会导致程序启动失败立刻终止，这时候请尝试重新运行，也许需要多试几次。
程序调用deepseek api-key，可以在platform.deepseek.com自行购买，初版没有做prompt上的优化，运行完成一次（5次问诊任务）需消耗价值约0.08元的tokens，api-key的格式为sk-xxx。当然如果你觉得麻烦，可以微信作者，我可以免费提供为这个项目测试所购买的api-key。
