# autofill_rjh_ai_tasks

一个用于辅助完成瑞金医院 AI 问诊任务的 Windows 小工具。

## 下载

- 仓库主页：[https://github.com/Young359/autofill_rjh_ai_tasks](https://github.com/Young359/autofill_rjh_ai_tasks)
- 下载 ZIP：[https://github.com/Young359/autofill_rjh_ai_tasks/archive/refs/heads/main.zip](https://github.com/Young359/autofill_rjh_ai_tasks/archive/refs/heads/main.zip)

## 使用方法

1. 下载并解压整个项目文件夹。
2. 双击运行 `全自动瑞金ai问诊_by_yqy.exe`。
3. 程序启动后会自动打开浏览器窗口。
4. 不要点击浏览器里的内容，尤其不要在浏览器中手动选择科室。
5. 需要选择科室或输入选项时，请按程序命令行窗口里的提示操作。

注意：不要只单独下载 `.exe` 文件运行，程序需要同目录下的 `_internal` 文件夹。

## 常见问题

如果网页加载较慢，程序可能启动失败。遇到这种情况，关闭程序后重新运行即可。

如果 Windows 提示未知发布者或安全提醒，可以先确认下载来源是本仓库，再选择继续运行。
